# 行为准则 (Code of Conduct)

## 我们的承诺

为了营造一个开放和友好的环境，我们作为项目的维护者和贡献者，承诺不论年龄、身体大小、残疾、种族、性别特征、性别认同和表达、经验水平、教育背景、社会经济地位、国籍、个人外貌、种族、宗教或性取向，都使每位参与者在我们的社区和项目中免受骚扰。

我们承诺以一种有助于建立一个健康、活跃和友好的开源社区的方式来开展工作和进行交流。

---

## 我们的标准

有助于为我们的社区创造积极环境的行为示例包括：

*   使用友好和包容的语言
*   尊重不同的观点和经验
*   礼貌地接受建设性的批评
*   关注对社区最有利的事情
*   对其他社区成员表示同理心和理解

参与者不可接受的行为示例包括：

*   使用性暗示的语言或意图，以及不受欢迎的性关注或调情
*   人身攻击、侮辱性/贬低性的言论
*   公开或私下的骚扰
*   未经明确许可，发布他人的私人信息，如物理地址或电子邮箱地址
*   其他在专业环境中可被合理视为不当的言行

---

## 维护者的职责

项目维护者有责任澄清可接受行为的标准，并被期望针对任何不可接受的行为采取适当且公正的纠正措施。

项目维护者有权利和责任删除、编辑或拒绝与本行为准则不符的 Commit、Code、Wiki、Issue 以及其他贡献，并对任何由于不当行为而暂时或永久禁言的贡献者进行沟通。

---

## 适用范围

本行为准则适用于所有与本项目相关的空间，包括但不限于官方的 GitHub 仓库、Issues、Pull Requests、讨论区，以及在代表项目或社区的公开或私人沟通场合。

---

## 执行与反馈

如果发现任何违反本行为准则的行为，或者受到不当行为的骚扰，请立即联系项目核心维护者 [小宝] (或通过 GitHub Issue 发起私密沟通)。所有投诉都将被迅速、认真地审查和调查。

项目维护者承诺对所有报告者及报告内容保持严格的机密性。
